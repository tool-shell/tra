# tra
**一个用基于nodejs的sheel翻译工具**

## 依赖
必须有nodejs版本在0.10.x以上都可以，npm必须的。另外在windows下暂时不知道是否有bug，有请提issue。

## 安装

```sh
$ sudo npm install -g tra
```
## 快速开始

安装完成后直接打开命令行输入`tra 单词`如输入`tra hello`
另外还有一些其他的：
* 输入`tra -h`打开帮助文档
* 输入`tra -v`显示版本

如果有问题，请提issue
